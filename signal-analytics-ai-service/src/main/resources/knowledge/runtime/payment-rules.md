# Regras de pagamento e carteira

Pagamento pendente significa que o pedido ainda aguarda processamento financeiro.

Pagamento pago significa que o pagamento foi aprovado.

Pagamento recusado pode estar relacionado a saldo insuficiente, dados inválidos ou inconsistência no pedido.

Saldo disponível é valor liberado para uso.

Saldo pendente é valor de venda aprovado, mas ainda aguardando liberação.

Para vendedores:
- saldo pendente pertence à loja
- saldo disponível da loja pode ser usado após liberação

Para clientes:
- saldo disponível representa valor para compras

Se houver pagamento recusado e não houver motivo claro, diga que os dados não trazem detalhes suficientes.
