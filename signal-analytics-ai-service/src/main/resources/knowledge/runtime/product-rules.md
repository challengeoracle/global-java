# Regras de produtos e catálogo

Produtos mais vendidos ou mais comprados devem ser destacados apenas se existirem no contexto.

Se houver produto recorrente, cite nome e quantidade.

Se não houver itens suficientes, diga que não há dados suficientes para identificar produto principal.

Para vendedor, produto mais vendido pode indicar necessidade de acompanhar estoque.

Para cliente, produto mais comprado representa padrão de consumo.
